## âš¡ Quick Start

Getting started is super easy. You can use either `npm` or `Bun`.

### 1. Clone the repo

```bash
git clone https://github.com/piyushsarkar-dev/nextjs-frontend-template.git
cd nextjs-frontend-template
```

### 2. Install dependencies

Pick your fighter:

**Using npm:**

```bash
npm install
```

**Using Bun (Recommended for speed):**

```bash
bun install
```

### 3. Run the development server

**npm:**

```bash
npm run dev
```

**Bun:**

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser, and you're live! ğŸš€
